Quick Start

1 Type Hebrew text — Use your keyboard or tap the on-screen keyboard. Text flows right-to-left automatically.
2 Add vowels — Click "Start Editing" to begin. The first Hebrew letter will be selected. Click any vowel to add it.
3 Navigate — Use Next/Prev buttons or arrow keys to move between letters. Use Skip to advance without adding vowels.
4 Edit specific letters — Select any letter in the text, then click "Start Editing" to edit from that position. Use "Clear Letter" to remove vowels.
5 Copy — Copy exports full text with vowels. No Vowels strips all vowels. Reverse flips the string order.

What are Nikkudot?

Nikkudot (also spelled Niqqud or Nekudot) are the diacritical marks used in Hebrew to indicate vowels and pronunciation. The Hebrew alphabet consists only of consonants, so nikkudot help readers pronounce words correctly.

While modern Hebrew texts typically omit nikkudot, they are essential for religious texts, poetry, children's books, and language learning materials.
